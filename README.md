# QT_game_Limited-time-arrest(双人娱乐放松小游戏————躲猫猫)
In this game, the player on the left is responsible for catching people, while the other side is responsible for hiding. Within the 50-second limit, if the left player catches someone, the game ends; if the right player successfully escapes, the game can restart.
在这个游戏中，左边的玩家负责捉人，而另一方负责躲藏。在50秒的时间限制内，如果左边的玩家抓住了某人，游戏结束，左边玩家为赢家；如果右边的玩家成功逃脱，则右边玩家赢得本场游戏，游戏可以重新开始。
结果可以商量着定
